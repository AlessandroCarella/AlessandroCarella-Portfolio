I worked on the analysis of a questionnaire carried by the editorial staff of Scomodo Bari. This project was created to support a collective writing article about aggregation spaces and urban social dynamics in the city of Bari. The work includes multiple data analysis studies conducted on survey responses from approximately 1,000 Bari residents.

The project transforms questionnaire data into actionable insights through five distinct analytical approaches: AI-powered citizen desires, regarding the city of Bari, using natural language processing analysis, geographic heatmap visualization of urban activity patterns both through classical heatmap representation, and with a neighboroods split, survey response categorization and visualization, and statistical correlation analysis of youth behavior patterns.

By combining machine learning techniques with interactive web visualizations, this project makes urban data accessible and contributes to evidence-based cultural analysis for the city of Bari,

Citizen Desires Analysis

Visualization

The visualization presents an AI-powered citizen engagement platform that analyzes open-ended survey responses from Bari residents about their vision for the city. It features two interactive views: a circle packing diagram and a treemap. Users can explore discovered topics through the interface that displays topic clusters, associated keywords, and allows drilling down into original citizen responses. The visualization provides users with an accessible way to understand what citizens care about most. Each topic is represented with proportional sizing based on frequency, and clicking on topics reveals the underlying text responses that contributed to each theme.

Notebook

The notebook implements Natural Language Processing techniques using BERTopic for unsupervised topic modeling on Italian text responses. It processes thousands of open-ended survey answers, cleaning and preprocessing the text data before applying transformer-based embeddings. The analysis automatically discovers thematic patterns in citizen responses without predefined categories, using dimensionality reduction and clustering algorithms to group similar responses. The notebook includes parameter tuning for optimal topic coherence, extraction of representative keywords for each topic, and preparation of structured data for the web visualization.

Geographic Heatmap

Visualization

This interactive geographic heatmap platform visualizes urban activity patterns across Bari neighborhoods using color-coded intensity zones. The interface allows users to toggle between satellite and standard map views, switch between different neighborhood boundary overlays, and adjust the heatmap radius for different levels of granularity. Features include smart masking to focus on specific geographic areas, and fractional zoom controls for precise navigation. The visualization uses a gradient color scheme to represent location frequency data, making spatial patterns immediately apparent. Users can hide or show the polygon overlay and neighborhood boundaries, and reset zoom to return to the full city view.

Notebook

The notebook handles geospatial data processing including coordinate validation, cleaning, and intensity calculations. It processes raw location data from survey responses, validates geographic coordinates to ensure they fall within Bari's boundaries, and calculates frequency metrics for each location. The analysis includes spatial aggregation techniques to generate density measurements across the city. The notebook exports structured datasets in formats compatible with Leaflet.js heatmap libraries, including coordinate pairs, intensity values, and neighborhood classification data. Additional processing includes handling edge cases, removing outliers, and preparing GeoJSON data for neighborhood boundary overlays.

Polygonal Heatmap - Neighborhoods Split

Visualization

Another version of the geographic heatmap that combines traditional heatmap visualization with polygon-based neighborhood analysis. This dual-layer approach allows users to visualize discrete neighborhood-level aggregations. The interface includes controls to toggle the polygon overlay, switch between neighborhood boundary versions, hide/show neighborhood names, and completely hide the heatmap layer to focus on neighborhood statistics. The visualization provides a more nuanced understanding of spatial patterns by showing how activity concentrates within specific administrative boundaries, while maintaining the granular detail of the continuous heatmap underneath.

This visualization shares the previous notebook since it is just another kind of visualization to the previously extracted data.

Survey Response Dashboard

Visualization

A modular web dashboard that presents survey analysis through multiple visualization modes including bar charts, pie charts, and treemaps. The interface features drag-and-drop reordering of chart elements, five customizable color themes to match different presentation contexts, and individual chart export functionality for creating reports. Users can switch between visualization types to explore the same data from different perspectives. The dashboard is designed to make complex survey insights accessible to non-technical stakeholders while maintaining analytical depth. Each visualization is interactive, with hover tooltips providing detailed statistics and percentages for individual data points.

Notebook

The notebook implements a full-stack survey data processing pipeline that categorizes question types (categorical, Likert scale , multiple-choice, multi-select) and applies appropriate cleaning and transformation techniques for each. It handles missing data, standardizes response formats, and implements one-hot encoding for categorical variables. The analysis includes frequency calculations, percentage computations, and statistical summaries for each survey question. The notebook exports structured JSON files organized by question type, with metadata about response options, labels, and statistical measures. This preprocessing ensures that the web dashboard receives clean, well-structured data optimized for D3.js visualization libraries.

Correlation Analysis

Visualization

A statistical visualization tool that presents correlation matrices as interactive, color-coded heatmaps. The interface displays relationships between various demographic, behavioral, and attitudinal variables from the youth survey data. Users can customize the color palette to highlight different correlation strengths, hover over cells to view exact correlation coefficients and statistical significance indicators, and export the visualization as PNG files for presentations. The heatmap uses color intensity to represent correlation strength, with different hues for positive and negative correlations. The visualization helps researchers and decision-makers quickly identify meaningful patterns and relationships in the survey data.

Notebook

This notebook implements data preprocessing techniques including one-hot encoding for categorical variables, multi-label binarization for questions where respondents could select multiple options, and proper handling of ordinal Likert scale data. It computes correlation matrices using appropriate methods (Pearson, Spearman, or Point-Biserial) based on variable types. The analysis identifies statistically significant correlations, filters out weak or non-significant relationships, and structures the results for visualization. The notebook includes detailed documentation of preprocessing decisions, handles edge cases like sparse variables, and generates JSON output containing the correlation matrix, variable labels, and metadata about statistical significance. This enables the web interface to display clear, scientifically rigorous correlation insights.
